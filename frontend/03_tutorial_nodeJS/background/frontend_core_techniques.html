一、Buffer

  • 1.概念

    • Buffer是一个类似于数组的对象，用于表示固定长度的字节序列
    • Buffer本质是一段内存空间，专门用来处理二进制数据

  • 2.特点

    • Buffer大小固定且无法调整
    • Buffer性能较好，可以直接对计算机内存进行操作
    • 每个元素的大小为1字节（Byte）

计算机的组成

  • CPU
  • 内存：读写快，断点丢数据
  • 硬盘：读写速度慢，断电不丢数据
  • 主板
  • 显卡

程序运行基本流程

  1. 操作系统
  2. 将程序载入内存
  3. CPU执行

程序一般保存在硬盘中，软件安装的过程就是将程序写入硬盘的过程。

程序在运行时会加载进内存，然后由CPU读取并执行程序

进程与线程

  • 进程

    • 进行中的程序
    • 是程序的一次执行过程

  • 线程

    • 线程是一个进程中执行的一个执行流
    • 一个线程是属于某个进程的
