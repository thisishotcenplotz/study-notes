为什么学Node.JS

  • 让更多人访问编写的网站
  • 为后续使用框架打好基础

Node.JS 定义

  • Node.JS is an open-source, cross-platform JavaScript runtime environment
  • 通俗定义：一款软件、应用程序可以运行JS代码

Node.JS 的作用

  1. 开发服务器应用
  2. 开发工具类应用
    • Webpack
    • Vite
    • Babel
  3. 开发桌面端应用
    • electron
      • VSCode
      • Figma
      • POSTMAN
