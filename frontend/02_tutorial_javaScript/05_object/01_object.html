数据类型：

  • 原始值
    1. 数值 Number
    2. 大整数 BigInt
    3. 字符串 String
    4. 布尔值 Boolean
    5. 空值 Null
    6. 未定义 Undefined
    7. 符号 Symbol
  • 对象
    • 对象是JS中的一种符合数据类型
    • 它就相当于一个容器，在对象中可以存储各种不同类型的数据
    • 对象中存储的数据，我们称为属性

原始值只能用来表示一些简单的数据，不能表示复杂数据

比如：现在需要再程序中表示一个人的信息

如果读取的事一个对象中没有的属性，JS不会报错会返回一个undefined
