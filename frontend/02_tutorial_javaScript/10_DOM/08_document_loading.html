网页是自上向下加载的，如果将JS代码编写在网页的上面，JS代码在执行时，网页还没有加载完毕，这时会出现无法获取DOM对象的情况


解决方案：

  1. 将script标签编写在body的最后边 --->(*****)
  2. 将代码编写到这个回调函数中：window.onload = function(){需要加载结束后执行的代码}
  3. 将代码编写到document对象的DocumentContentLoaded的回调函数中(执行时机更早)
  4. 将代码编写到外部JS文件中，然后以defer的形式引用 (这种执行机制更早，早于上面那种) ---> (开发中最常使用) --->(*****)
