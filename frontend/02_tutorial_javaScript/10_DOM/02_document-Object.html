Document对象

  • Document对象表示的是整个网页
  • Document对象的原型链
    • HTMLDocument
    • Document
    • Node
    • EventTarget
    • Object.prototype
    • null
  • 凡是在原型链上存在的对象的属性和方法都可以通过document去调用
  • 部分属性：
    • document.documentElement ---> html根元素
    • document.head ---> head元素
    • document.title ---> title元素
    • document.body ---> body元素
    • document.links ---> 获取网页中所有超链接
