将其他类型的数据转换为数值

  1. 使用Number()函数来将其他类型转换为数值
    • 数字字符串：如果字符串是一个合法的数字，则会转换为对应的数字
    • 字母字符串：不合法数字将会转为NaN
    • 空字符串：会转换为0
    • 空格字符串：会转换为0
    • 布尔值：true--->1；false--->0;
    • null: 转换为0
    • undefined：NaN
  2. 专门用来将字符串转换为数值的两个方法
    • parseInt():从头开始取有效数字，如果开头不是数字则无法转换
      • 可以临时使用parseInt()来进行取整，但不是最优解。
    • parseFloat()：从头开始取有效小数，如果开头不是数字则无法转换
