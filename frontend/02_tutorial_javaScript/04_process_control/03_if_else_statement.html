if-else

  • 执行流程：
    • if-else执行时，先对条件表达式进行求值判断
    • 如果为true则执行if后的语句
    • 如果为false，则执行else后的语句
  • if-else if-else:
    • if-else if-else语句会自上向下依次对if后的语句的条件表达式进行求值判断
    • 如果条件表达式结果为true，则执行当前if后的语句，执行完毕语句结束
    • 如果条件表达式为false，则继续向下判断，知道找到true为止
    • 如果所有的条件表达式都是false，则执行else后的语句
  • 注意：if-else if -else 语句中只会有一个代码块被执行，一旦有执行的代码块，下边的条件都不会继续判断了，所以要注意条件编写的顺序
