对象的序列化

  • JS中的对象使用时都存在于计算机的内存中
  • 序列化指将对象转化为一个可以存储的格式
  • 在JS中对象的序列化通常是将一个对象转化为字符串
  • 序列化的作用：
    1. 对象转化为字符串后，可以将字符串在不同的语言之间进行传递，甚至可以直接对字符串进行读写操作，使得JS对象可以在不同语言之间传递
    2. 用途：
      • 作为数据交换的格式
      • 用来编写配置文件
  • 如何进行序列化：
    • 在JS中有一个工具类JSON（javascript object notation）
    • JS对象序列化后会变成一个字符串，这个字符串就叫JSON
  • 也可以手动编写JSON，在很多程序的配置文件中就是使用的JSON
  • 编写JSON注意事项：
    • JSON字符串有两种类型： JSON对象 {} 或 JSON数组 []
    • JSON字符串的属性名必须使用双引号
    • JSON中可以使用的属性值和元素
      1. 数字（Number）
      2. 字符串（String）必须使用双引号
      3. 布尔值（Boolean）
      4. 空值（Null）
      5. 对象（Object {}）
      6. 数组（Array []）
  • JSON的格式和JS对象的格式基本上一致，但是JSON字符串如果属性时最后一个，则不加逗号
